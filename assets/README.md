# Assets de marca (S.G.I.)

Archivos de identidad visual usados por el empaquetado:

- `logo.png` — logo oficial de S.G.I. (PNG, idealmente ≥512×512 y fondo
  transparente). **Colócalo aquí**; de él se genera el icono.
- `icon.ico` — icono multi-resolución (16/32/48/256) generado desde `logo.png`.
  El spec de PyInstaller lo toma automáticamente si existe.
